# Dependancies

● wine 7

● wget

# How to run?
```
git clone https://github.com/Caliel666/DivaModManager.git
cd DivaModManager
chmod +x divamodmanager-linux.sh
./divamodmanager-linux.sh
```

after running the installer you should have DivaModManager on your applications menu
